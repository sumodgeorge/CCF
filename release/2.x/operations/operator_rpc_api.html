Operator RPC API#

Operator RPCs are exposed under /node and do not require authentication.

GET /node/api#

Example request:

GET /node/api HTTP/1.1
Host: example.com
Status Codes

  • 200 OK –

    Default response description

    Example response:

    HTTP/1.1 200 OK
Content-Type: application/json

{}
