◆ read_key()

const MapHandle::ValueType * kv::untyped::MapHandle::read_key ( const KeyType &  key)
protected

Get pointer to current value if this key exists, else nullptr if it does not exist or has been deleted. If non-null, points to something owned by tx_changes - expect this is used/dereferenced immediately, and there is no concurrent access which could invalidate it. Modifies read set if appropriate to record read dependency on this key, at the version of the returned data.
